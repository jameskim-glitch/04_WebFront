CSS(Cascading Style Sheets)

    마크업 언어(HTML, XML)가 실제로 화면에 표시되는 방법(모양, 색상, 크기, 스타일... )을 기술하는 언어
    - HTML로 작성된 문서의 레이아웃, 색상, 크기, 폰트, 효과 등을 꾸밀 때 사용
    


    
CSS Selector(선택자)


    선택자란? 
    현재 html 문서 내에서 특정한 요소를 선택하는 문자 또는 문자열
    

    스타일 적용하고자 하는 요소를 선택하는 방법

태그 선택자

    현재 HTML 문서 내에 같은 태그를 선택하는 선택자

    [작성법]

    태그명 {
      css 코드;
    }
Strong 태그에 작성된 내용
Span 태그에 작성된 내용
  • Strong 태그에 작성된 내용
  • Span 태그에 작성된 내용

id 선택자 (#)

  html 문서 내에서 id 속성값이 일치하는 요소를 선택하는 선택자

  * id 속성 : 요소를 식별할 수 있는 고유 식별자 역할
  한 html 문서 내에서 id 속성값은 유일해야 함

  [작성법]

  #id 속성값{
    css 코드;
  }

  # == id 선택자를 의미
  1. id 선택자 1
  2. id 선택자 1
  3. id 선택자 1
  4. id 선택자 1
  5. id 선택자 1

class 선택자(.)

      html 문서 내에서 class 속성값이 일치하는 요소를 선택하는 선택자

      * class 속성 : 여러 요소에 같은 스타일을 적용할 때 사용
      한 html 문서 내에서 class 속성값은 중복 가능

      [작성법]

      .class 속성값{
        css 코드;
      }

      . == class 선택자를 의미

모든 요소 선택자(*)

      html 문서 내의 모든 요소를 선택하는 선택자

      [작성법]

      *{
        css 코드;
      }
