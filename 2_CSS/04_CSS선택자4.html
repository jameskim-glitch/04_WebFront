일반 구조 선택자

    형제 관계에 있는 요소 중 특정 요소를 선택하는 선택자

    위치를 기준으로 구분함(중요)

    [작성법]
    선택자1 + 선택자2 {
      css 코드;
    }
    :firdt-child : 첫번째 요소를 선택
    :last-child : 마지막 요소를 선택
    :nth-child(n) : n번째 요소를 선택(수열)
    :nth-last-child(n) : 뒤에서 n번째 요소를 선택(수열)
    :nth-of-type(n) : n번째 요소를 선택

형태 구조 선택자

    선택된 형제관계 요소중 특정 요소를 선택하는 선택자

    (선택된 요소들을 기준으로 구분)

    :first-of-type : 첫번째 요소를 선택
    :last-of-type : 마지막 요소를 선택
    :nth-of-type(n) : n번째 요소를 선택(수열)
    :nth-last-of-type(n) : 뒤에서 n번째 요소를 선택(수열)
