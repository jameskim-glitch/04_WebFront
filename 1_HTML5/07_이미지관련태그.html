img 태그

    - 웹 문서에 사진, 그림 같은 이미지를 삽입하는 용도의 태그
    [속성]
    - src: 이미지 파일의 경로 를 지정하는 속성(파일이 있는 경로, 웹 주소)
    - alt: 이미지 파일이 없을 때 대체 텍스트를 지정하는 속성(이미지 설명)
    - width: 이미지의 너비를 지정하는 속성(픽셀 단위)
    - height: 이미지의 높이를 지정하는 속성(픽셀 단위)
    - border: 이미지 주변에 테두리를 지정하는 속성(픽셀 단위)
    - usemap: 이미지를 클릭 가능한 맵으로 지정하는 속성
    - ismap: 이미지를 클릭 가능한 맵으로 지정하는 속성
    - align: 이미지의 정렬을 지정하는 속성(left, right, top, middle, bottom)
    - hspace: 이미지의 좌우 여백을 지정하는 속성(픽셀 단위)
    - vspace: 이미지의 상하 여백을 지정하는 속성(픽셀 단위)
    - title: 이미지에 대한 설명을 지정하는 속성(툴팁)
    - longdesc: 이미지에 대한 상세 설명을 지정하는 속성(웹 주소)
    - style: 이미지에 대한 스타일을 지정하는 속성
    - class: 이미지에 대한 스타일을 지정하는 속성
    - id: 이미지에 대한 스타일을 지정하는 속성
    - data-*: 사용자 정의 데이터 속성을 지정하는 속성
    - draggable: 이미지를 드래그 가능하게 지정하는 속성
    - ondragstart: 이미지를 드래그 시작할 때 이벤트를 지정하는 속성
    - ondragend: 이미지를 드래그 끝날 때 이벤트를 지정하는 속성
    - ondrag: 이미지를 드래그 중일 때 이벤트를 지정하는 속성
    - ondragenter: 이미지를 드래그할 때 들어갈 때 이벤트를 지정하는 속성
    - ondragleave: 이미지를 드래그할 때 나갈 때 이벤트를 지정하는 속성
    - ondragover: 이미지를 드래그할 때 위에 있을 때 이벤트를 지정하는 속성
    - ondrop: 이미지를 드래그할 때 놓을 때 이벤트를 지정하는 속성

src 속성 확인

width, height 속성을 이용한 크기 조절(고정/가변 크기)

고정 크기 단위(px) 화면의 크기와 상관 없이 지정된 값의 크기를 갖는 단위

가변 크기 단위(%) 화면이나 영역 기준으로 사이트 크기에 따라 이미지의 크기가 변경되는 단위
