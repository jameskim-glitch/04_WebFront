영역을 나누는 형식(성질)

    
block 형식

    - 항상 새로운 줄에서 시작!
    - 이전 요소의 아래에 위치(블럭이 아래로 쌓임)
    - 너비, 높이 지정 가능
    - width: 100% (기본값)  -> 부모 요소의 너비를 100%로 설정
    - height: auto (기본값) -> 자식 요소의 높이에 따라 늘어남 (자식 요소의 높이가 없으면 0)
    - margin, padding, border 속성을 사용할 수 있음 (margin: 0 auto; -> 가운데 정렬)
    - div, h1, h2, h3, h4, h5, h6, p, ul, ol, li, dl, dt, dd, table, form, fieldset, address, blockquote, pre, hr, noscript, header, footer, nav, article, section, aside, main, figure, figcaption, div, span

    ex) p/pre (문단 나누기), h1~h6(제목 영역 나누기), hr (수평선 넣어서 화면 나누기), div (영역(divison) 나누기) 
    
block

  
    
inline 형식

    - 새로운 줄에서 시작하지 않음
    - 이전 요소의 옆에 위치(인라인이 옆으로 쌓임)
    - 너비, 높이 지정 불가능
    - width: auto (기본값) -> 자식 요소의 너비에 따라 늘어남 (자식 요소의 너비가 없으면 0)
    - height: auto (기본값) -> 자식 요소의 높이에 따라 늘어남 (자식 요소의 높이가 없으면 0)
    - margin, padding, border 속성을 사용할 수 있음 (margin: 0 auto; -> 가운데 정렬 불가능)
    - span, a, img, strong, em, b, i, u, s, sub, sup, br, input, select, textarea, button, label, code, kbd, samp, var, cite, abbr, acronym, q, dfn, small, big, tt, strike, font, basefont, bdo, script, style, link, meta, title, base, head, body, html

    ex) a (링크), img (이미지), span (텍스트 일부분), strong/em (강조), br (줄바꿈), input/select/textarea/button (입력 양식), label (입력 양식 설명), code/kbd/samp/var/cite/abbr/acronym/q/dfn/small/big/tt/strike/font/basefont/bdo (텍스트 스타일), script/style/link/meta/title/base/head/body/html (문서 구조)
