하이퍼링크 관련 태그 : a(anchor) 태그

    - 웹 문서에서 다른 웹 문서로 이동할 수 있는 링크를 만들 때 사용하는 태그
    - a 태그(anchor 태그)를 사용하여 링크를 만들 수 있음
    - href 속성을 사용하여 이동할 웹 문서의 주소를 지정
    - target 속성을 사용하여 링크를 클릭했을 때 이동할 방법을 지정
    - title 속성을 사용하여 링크에 대한 설명을 지정
    - download 속성을 사용하여 링크를 클릭했을 때 다운로드할 수 있도록 지정
    - rel 속성을 사용하여 링크와 현재 문서와의 관계를 지정
    - type 속성을 사용하여 링크된 문서의 MIME 타입을 지정
    - hreflang 속성을 사용하여 링크된 문서의 언어를 지정
    - media 속성을 사용하여 링크된 문서의 미디어 유형을 지정
    - charset 속성을 사용하여 링크된 문서의 문자 인코딩 방식을 지정
    - rev 속성을 사용하여 링크와 현재 문서와의 관계를 지정
    - shape 속성을 사용하여 링크된 문서의 모양을 지정
    - coords 속성을 사용하여 링크된 문서의 좌표를 지정
    - name 속성을 사용하여 링크된 문서의 이름을 지정
    - accesskey 속성을 사용하여 링크에 단축키를 지정
    - tabindex 속성을 사용하여 링크에 탭 순서를 지정
    - onfocus 속성을 사용하여 링크에 포커스가 맞춰졌을 때 이벤트를 지정
    - onblur 속성을 사용하여 링크에 포커스가 빠져나갔을 때 이벤트를 지정
    - onmouseover 속성을 사용하여 마우스가 링크에 올려졌을 때 이벤트를 지정
  
    a 태그는 현재 문서와 다른 웹 문서로 이동할 수 있는 링크를 만들 때 사용하는 태그입니다.
    a 태그(anchor 태그)를 사용하여 링크를 만들 수 있습니다.
    a 태그로 감싸진 문자열 또는 이미지를 클릭하면 하이퍼링크 속성에 지정된 웹 문서로 이동합니다.
    a 태그는 다음과 같은 속성을 사용할 수 있습니다.
    - href 속성: 이동할 웹 문서의 주소를 지정합니다.
    - target 속성: 링크를 클릭했을 때 이동할 방법을 지정합니다.
    - title 속성: 링크에 대한 설명을 지정합니다.
    - download 속성: 링크를 클릭했을 때 다운로드할 수 있도록 지정합니다.
    - rel 속성: 링크와 현재 문서와의 관계를 지정합니다.
    - type 속성: 링크된 문서의 MIME 타입을 지정합니다.
    - hreflang 속성: 링크된 문서의 언어를 지정합니다.
    - media 속성: 링크된 문서의 미디어 유형을 지정합니다.
    - charset 속성: 링크된 문서의 문자 인코딩 방식을 지정합니다.
    - rev 속성: 링크와 현재 문서와의 관계를 지정합니다.
    - shape 속성: 링크된 문서의 모양을 지정합니다.
    - coords 속성: 링크된 문서의 좌표를 지정합니다.
    - name 속성: 링크된 문서의 이름을 지정합니다.
    - accesskey 속성: 링크에 단축키를 지정합니다.
    - tabindex 속성: 링크에 탭 순서를 지정합니다.
    - onfocus 속성: 링크에 포커스가 맞춰졌을 때 이벤트를 지정합니다.
    - onblur 속성: 링크에 포커스가 빠져나갔을 때 이벤트를 지정합니다.
    - onmouseover 속성: 마우스가 링크에 올려졌을 때 이벤트를 지정합니다.
