입력 관련 태그 : input 태그, form 태그

    - 웹 문서에서 사용자로부터 입력을 받을 때 사용하는 태그
    - input 태그를 사용하여 입력 양식을 만들 수 있음
    - form 태그를 사용하여 입력 양식을 묶을 수 있음
    - input 태그는 다양한 속성을 사용하여 입력 양식을 만들 수 있음
    - form 태그는 다양한 속성을 사용하여 입력 양식을 묶을 수 있음
    - input 태그는 다음과 같은 속성을 사용할 수 있음
    - type 속성: 입력 양식의 종류를 지정
    - name 속성: 입력 양식의 이름을 지정
    - value 속성: 입력 양식의 값을 지정
    - placeholder 속성: 입력 양식에 힌트를 지정
    - required 속성: 입력 양식이 필수로 입력되어야 하는지 지정
    - disabled 속성: 입력 양식이 비활성화되어 있는지 지정
    - readonly 속성: 입력 양식이 읽기 전용인지 지정
    - size 속성: 입력 양식의 크기를 지정
    - maxlength 속성: 입력 양식에 입력할 수 있는 최대 길이를 지정
    - min 속성: 입력 양식에 입력할 수 있는 최소값을 지정
    - max 속성: 입력 양식에 입력할 수 있는 최대값을 지정
    - step 속성: 입력 양식에 입력할 수 있는 값의 간격을 지정
    - pattern 속성: 입력 양식에 입력할 수 있는 값의 패턴을 지정
    - autocomplete 속성: 입력 양식에 자동완성 기능을 사용할지 지정
    - form 속성: 입력 양식이 속한 form 태그의 id를 지정
    - formaction 속성: 입력 양식이 제출 될 때 이동할 웹 문서의 주소를 지정
    - formenctype 속성: 입력 양식이 제출 될 때 서버로 전송할 데이터의 유형을 지정

form 태그

        
        - div와 같은 영역(block 형식)
        - 내부에 작성된 input 태그의 값을 서버로 전달하는 역할


        [속성]
        1) action : 내부에 작성된 값을 전달할 서버 또는 페이지 주소를 작성
        (미작성 시 기본값으로 현재 페이지의 URL 적용)
        2) name : form 태그의 이름을 지정하여 form 태그 구분(JS 사용)
        3) method(방식) : 서버로 데이터 전달하는 방식을 지정하는 속성
        - GET / POST 지정 가능(기본값 GET)
        4) target : action에 지정된 주소를 현재탭/새탭 중 어디서 열지 지정
